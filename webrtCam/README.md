# webrtc cam
유튜브를 보며 만들어 보는 프로젝트
https://www.youtube.com/watch?v=DvlyzDZDEq4

### npm init
```bash
npm init -y
```

### npm dependencies
```bash
npm i express ejs socket.io
```

### nodemon install
```bash
npm i -- save-dev nodemon
```

### npm peer install
```bash
npm i -g peer
or
npm i peer
```

### Dev Start
package.json
add script
"devStart": "nodemon server.js"

```bash
npm run devStart
```
